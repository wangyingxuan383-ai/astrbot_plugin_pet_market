# 🐾 AstrBot 插件：宠物市场

把群友变成宠物，开启养成、打工、训练、银行、PK决斗、进化、排行榜的完整互动游戏！

![AstrBot](https://img.shields.io/badge/AstrBot-v4.5.7+-blue)
![Version](https://img.shields.io/badge/Version-v2.1.0-green)

---

## ✨ 功能特性

- 🛒 **购买/放生宠物** - 把群友变成你的宠物
- 💼 **打工赚钱** - 派遣宠物打工获取金币
- 📈 **训练升值** - 提升宠物身价
- ⭐ **进化系统** - 普通→稀有→史诗→传说
- ⚔️ **PK 决斗** - 宠物对战，赢家掠夺 10% 身价
- 🏦 **银行系统** - 存款、取款、复利利息
- 💸 **转账系统** - 玩家间转账（含手续费）
- 💣 **抢劫系统** - 风险与收益并存
- 📊 **排行榜** - 身价/资金/总资产排名（支持分页）

---

## 🎮 用户指令

| 指令 | 说明 |
|------|------|
| `/宠物菜单` | 查看所有可用指令（图片） |
| `/宠物市场 [页码]` | 查看群内宠物列表（支持分页） |
| `/购买宠物 @群友` | 购买指定群友为宠物 |
| `/放生宠物 @群友` | 放生宠物 |
| `/打工` | 派遣宠物打工赚钱 |
| `/训练 @群友` | 训练宠物提升身价 |
| `/进化宠物 @群友` | 消耗金币进化宠物 |
| `/PK @群友` | ⚔️ 宠物决斗（赢家掠夺10%身价） |
| `/我的宠物` | 查看自己的宠物与金币 |
| `/银行信息` | 查看银行等级与利息 |
| `/升级信用` | 提升银行等级与存储上限 |
| `/领取利息` | 领取银行存款利息 |
| `/存款 金额` | 存入金币到银行 |
| `/取款 金额` | 从银行取出金币 |
| `/转账 @群友 金额` | 转账给其他玩家 |
| `/转账记录` | 查看最近10条转账记录 |
| `/宠物身价排行榜 [页码]` | 查看身价排行（支持分页） |
| `/宠物资金排行榜 [页码]` | 查看余额排行（支持分页） |
| `/群内十大首富 [页码]` | 查看总资产排行（支持分页） |
| `/抢劫 @群友` | 抢劫其他玩家 |

## 🔧 管理员指令

| 指令 | 说明 |
|------|------|
| `/我发钱 金额` | 给自己发金币（调试） |

## ⚙️ 配置说明（WebUI）

| 配置项 | 默认值 | 说明 |
|--------|--------|------|
| `work_cooldown` | 3600 秒 | 打工冷却时间 |
| `train_cooldown` | 86400 秒 | 训练冷却时间 |
| `pk_cooldown` | 3600 秒 | PK 决斗冷却时间 |
| `rob_cooldown` | 3600 秒 | 抢劫冷却时间 |
| `transfer_cooldown` | 1800 秒 | 转账冷却时间 |
| `train_success_rate` | 0.7 | 训练成功率 |
| `bank_interest_rate` | 0.01 | 银行每小时利息率（复利） |
| `transfer_fee_rate` | 0.1 | 转账手续费率 |

---

## 📥 安装方式

1. 将本插件文件夹放入 AstrBot 的 `plugins/` 目录
2. 重启 AstrBot
3. 群聊发送 `/宠物菜单` 开始游戏
